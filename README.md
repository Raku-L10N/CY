NAME
====

L10N::CY - Welsh localization of Raku

SYNOPSIS
========

```raku
use L10N::CY;
dywedyd "Hello World";
```

DESCRIPTION
===========

L10N::CY contains the logic to provide a Welsh localization of the Raku Programming Language.
